# 使用方法

1. clone代码到本地

2. 打开命令行并cd到代码目录

3. 使用 `composer install`

4. 使用 `yarn install`

5. 将 <b>.env.example</b>文件名改为 <b>.env</b>并将

   ```
   DB_CONNECTION=mysql
   DB_HOST=127.0.0.1
   DB_PORT=3306
   DB_DATABASE=laravel
   DB_USERNAME=root
   DB_PASSWORD=
   
   ```

   字段改为本地数据库相关信息

6. 命令行中输入`php artisan serve`即可启动服务器
